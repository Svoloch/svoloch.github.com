Подсветка текста.

Модуль представляет собой расширение xpath-tools в виде двух методов позволяющих добавлять и убирать подсветку. А так же два метода используемые основными методами.


$X.Class::highlight :: (Str, Str)->$X.Class #Полусчает класс подсветки и подсвечиваемый текст, возвращает элементы с подсветкой.
$X.Class::unspan :: (Str)->$X.Class #Убирает <span/> теги c заданым классом, так удобно убирать подсветку.
$X.Class::unwrap :: ->Null #Заменяет тег на его содержимое.
$X.Class::normalizeText :: ->$X.Class #Сливает соседствующие текстовые узлы в один. Возвращает this.
